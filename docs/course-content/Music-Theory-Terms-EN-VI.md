### Module 1: Nền tảng (Cao độ \& ký âm)

| Thuật ngữ tiếng Việt | Thuật ngữ tiếng Anh | Ghi chú ngắn |
| :-- | :-- | :-- |
| Khuông nhạc | Staff / Stave | $5$ dòng kẻ để viết nốt |
| Dòng kẻ (khuông) | Lines (of the staff) | Các line 1–5 |
| Khe (khuông) | Spaces (of the staff) | Các khoảng giữa dòng |
| Khóa nhạc | Clef | Xác định vùng cao độ trên khuông |
| Khóa Sol | Treble Clef / G Clef | Thường cho giọng cao, tay phải piano, guitar… |
| Khóa Fa | Bass Clef / F Clef | Thường cho giọng trầm, tay trái piano, bass… |
| Tên nốt nhạc | Note names | $A\text{-}B\text{-}C\text{-}D\text{-}E\text{-}F\text{-}G$ |
| Cao độ | Pitch | “Độ cao” của âm thanh |
| Nốt nhạc (ký hiệu) | Note (symbol) | Hình nốt trên khuông (khác “pitch”) |
| Bàn phím (piano) | Keyboard | Đối chiếu vị trí nốt |
| Dấu thăng | Sharp $(\#)$ | Tăng $1$ bán âm |
| Dấu giáng | Flat $(\flat)$ | Giảm $1$ bán âm |
| Dấu bình (dấu tự nhiên) | Natural $(\natural)$ | Trả về nốt “tự nhiên” |
| Dấu hóa | Accidentals | Nhóm: $\#$, $\flat$, $\natural$ |
| Cung (nguyên âm) | Whole step / Whole tone | Bước $2$ bán âm |
| Nửa cung (bán âm) | Half step / Semitone | Bước $1$ bán âm |
| Nốt đồng âm khác tên | Enharmonic equivalents | Ví dụ $C\#$ và $D\flat$ |

### Module 2: Nhịp điệu \& nhịp (Rhythm \& Meter)

| Thuật ngữ tiếng Việt | Thuật ngữ tiếng Anh | Ghi chú ngắn |
| :-- | :-- | :-- |
| Trường độ nốt | Note values / Note durations | Độ dài của nốt |
| Nốt tròn | Whole note | Thường tính $4$ phách trong $4/4$ |
| Nốt trắng | Half note | $2$ phách trong $4/4$ |
| Nốt đen | Quarter note | $1$ phách trong $4/4$ |
| Nốt móc đơn | Eighth note | $1/2$ phách trong $4/4$ |
| Nốt móc kép | Sixteenth note | $1/4$ phách trong $4/4$ |
| Dấu lặng | Rest(s) | Khoảng im lặng có trường độ |
| Lặng tròn | Whole rest | Tương ứng nốt tròn |
| Lặng trắng | Half rest | Tương ứng nốt trắng |
| Lặng đen | Quarter rest | Tương ứng nốt đen |
| Lặng móc đơn | Eighth rest | Tương ứng nốt móc đơn |
| Lặng móc kép | Sixteenth rest | Tương ứng nốt móc kép |
| Nốt chấm dôi | Dotted note | Tăng thêm $1/2$ giá trị nốt |
| Dấu nối | Tie | Nối trường độ 2 nốt cùng cao độ |
| Nhịp | Meter | Cách chia phách theo ô nhịp |
| Ô nhịp | Bar / Measure | Đơn vị chứa số phách theo nhịp |
| Vạch nhịp | Barline | Vạch phân chia ô nhịp |
| Chỉ số nhịp | Time signature | Ví dụ $4/4, 3/4, 2/4$ |
| Tử số / mẫu số (chỉ số nhịp) | Top number / Bottom number | Tử: số phách; mẫu: loại nốt được 1 phách |
| Nhịp đơn | Simple meter | Chia phách theo $2$ (thường) |
| Nhịp kép / nhịp phức | Compound meter | Chia theo $3$; ví dụ $6/8$ |
| Cảm giác bộ ba | Triplet feel | “Lắc” theo bộ $3$ |
| Bộ ba | Triplet | Chia đều 1 phách thành $3$ |
| Tốc độ | Tempo | Nhanh/chậm của nhạc |
| Nhịp mỗi phút | BPM (Beats Per Minute) | Số phách/phút |
| Ký hiệu tốc độ Ý | Italian tempo markings | Ví dụ Largo, Andante, Allegro |
| Largo | Largo | Rất chậm |
| Andante | Andante | Vừa chậm, “đi bộ” |
| Allegro | Allegro | Nhanh, vui |

### Module 3: Thang âm \& giai điệu (Scales \& Melody)

| Thuật ngữ tiếng Việt | Thuật ngữ tiếng Anh | Ghi chú ngắn |
| :-- | :-- | :-- |
| Thang âm | Scale | Chuỗi nốt theo quy luật |
| Giai điệu | Melody | Chuỗi nốt theo thời gian tạo “câu hát” |
| Thang âm trưởng | Major scale | Công thức: $W\text{-}W\text{-}H\text{-}W\text{-}W\text{-}W\text{-}H$ |
| Công thức cung–nửa cung | Whole/Half-step formula | $W$=cung, $H$=nửa cung |
| Giọng / điệu (tông) | Key | Ví dụ giọng $C$ trưởng |
| Dấu hóa ở khóa (dấu hóa cố định) | Key signature | Thăng/giáng đặt đầu khuông |
| Thứ tự dấu thăng | Order of sharps | $F\#\ C\#\ G\#\ D\#\ A\#\ E\#\ B\#$ |
| Thứ tự dấu giáng | Order of flats | $B\flat\ E\flat\ A\flat\ D\flat\ G\flat\ C\flat\ F\flat$ |
| Quãng | Interval | Khoảng cách giữa 2 nốt |
| Bậc quãng (số quãng) | Interval quantity / Number | 2nd, 3rd, 4th, 5th… |
| Chất lượng quãng | Interval quality | Major/Minor/Perfect/Aug/Diminished |
| Quãng trưởng | Major interval | “Trưởng” |
| Quãng thứ | Minor interval | “Thứ” |
| Quãng đúng | Perfect interval | Thường cho 4, 5, 8… |
| Quãng tăng | Augmented interval | Nới rộng thêm $1$ bán âm |
| Quãng giảm | Diminished interval | Thu hẹp $1$ bán âm |
| Thang âm thứ | Minor scales | Nhóm các dạng thứ |
| Thứ tự nhiên | Natural minor | Dạng cơ bản |
| Thứ hòa âm | Harmonic minor | Nâng bậc $7$ |
| Thứ giai điệu | Melodic minor | Thường nâng bậc $6,7$ khi đi lên |
| Thang âm ngũ cung | Pentatonic scale | $5$ nốt, rất hay dùng để solo |
| Ngũ cung trưởng | Major pentatonic | Mẫu ngũ cung trưởng |
| Ngũ cung thứ | Minor pentatonic | Mẫu ngũ cung thứ |

### Module 4: Hòa âm (Trọng tâm)

| Thuật ngữ tiếng Việt | Thuật ngữ tiếng Anh | Ghi chú ngắn |
| :-- | :-- | :-- |
| Hòa âm | Harmony | Cách các nốt/chord chồng lên nhau |
| Hợp âm | Chord | Nhiều nốt vang cùng lúc |
| Hợp âm ba | Triad | $3$ nốt: gốc–3–5 |
| Nốt gốc | Root | Nốt “tên” của hợp âm |
| Quãng 3 | Third | Bậc tạo tính trưởng/thứ |
| Quãng 5 | Fifth | Tạo độ vững của triad |
| Chất hợp âm | Chord quality | Major/Minor/Dim/Aug… |
| Hợp âm trưởng | Major triad | Công thức quãng $1\text{-}3\text{-}5$ kiểu trưởng |
| Hợp âm thứ | Minor triad | Kiểu thứ |
| Hợp âm giảm | Diminished triad | Quãng 5 giảm |
| Hợp âm tăng | Augmented triad | Quãng 5 tăng |
| Hợp âm trong giọng (hợp âm điệu tính) | Diatonic chords | Chỉ dùng nốt thuộc 1 giọng |
| Phân tích số La Mã | Roman numeral analysis | $I, ii, iii, IV, V, vi, vii^\circ$ |
| Hệ thống số Nashville | Nashville Number System | Ghi hợp âm theo bậc trong giọng |
| Vòng tròn quãng 5 | Circle of fifths | Quan hệ giọng, dấu hóa |
| Đảo hợp âm | Chord inversions | Đổi nốt bass: root/1st/2nd inversion |
| Hợp âm gạch chéo | Slash chord | Ví dụ $C/E, G/B$ |
| Dẫn giọng | Voice leading | Chuyển động các bè mượt |
| Hợp âm bảy | Seventh chords | Thêm nốt bậc $7$ lên triad |
| Trưởng bảy | Major 7 (Maj7) | Ví dụ $C\mathrm{Maj}7$ |
| Thứ bảy | Minor 7 (min7) | Ví dụ $D\mathrm{m}7$ |
| Bảy át (bảy trội) | Dominant 7 (dom7 / 7) | Ví dụ $G7$ dẫn về $C$ |

### Module 5: Sáng tác \& hình thức (Composition \& Form)

| Thuật ngữ tiếng Việt | Thuật ngữ tiếng Anh | Ghi chú ngắn |
| :-- | :-- | :-- |
| Sáng tác | Composition | Viết nhạc |
| Hình thức | Form | Cấu trúc bài |
| Tiến trình hợp âm | Chord progression | Chuỗi hợp âm theo thời gian |
| Tiến trình phổ biến | Common chord progressions | Ví dụ $I\text{-}V\text{-}vi\text{-}IV$, $ii\text{-}V\text{-}I$ |
| Kết (cadence) | Cadence | “Dấu câu” của hòa âm |
| Kết xác thực | Perfect cadence / Authentic cadence | Thường $V\rightarrow I$ |
| Kết nhà thờ | Plagal cadence | Thường $IV\rightarrow I$ (“Amen”) |
| Kết lửng | Half cadence | Dừng ở $V$ |
| Kết tránh | Deceptive cadence | Thường $V\rightarrow vi$ |
| Đường nét giai điệu | Melodic contour | Hướng đi lên/xuống của giai điệu |
| Âm lướt | Passing tone | Nốt “đi qua” nối 2 nốt chính |
| Âm láng giềng | Neighbor tone | Đi ra rồi quay về nốt chính |
| Cấu trúc bài hát | Song structure | Verse/Chorus/Bridge… |
| Đoạn phiên khúc | Verse | Kể chuyện, thay lời |
| Điệp khúc | Chorus | Phần “hook”, lặp |
| Cầu / đoạn chuyển | Bridge | Tạo tương phản |
| Tiền điệp khúc | Pre-chorus | Dẫn vào chorus |
| Dạo đầu | Intro | Mở đầu |
| Kết / dạo cuối | Outro | Kết thúc |
| Cường độ | Dynamics | To/nhỏ |
| Ký hiệu cường độ | Dynamic markings | $p, f, mf$… |
| Nhẹ | $p$ (piano) | Nhỏ |
| Vừa | $mf$ (mezzo-forte) | Vừa |
| Mạnh | $f$ (forte) | To |
| Phát âm/nhấn | Articulation | Cách đánh/nhả nốt |
| Ngắt (rời tiếng) | Staccato | Ngắn, tách |
| Liền tiếng | Legato | Mượt, nối |
| Chuyển giọng | Modulation | Đổi key trong bài (thường tạm thời) |

Nếu bạn muốn, mình có thể chuẩn hóa theo một “chuẩn Việt” cụ thể (ví dụ: dùng “giọng” hay “điệu”, “kết” hay “kết câu”), hoặc bổ sung cột viết tắt/ký hiệu hay gặp trên sheet (Maj, min, dim, aug, sus, add9…).
